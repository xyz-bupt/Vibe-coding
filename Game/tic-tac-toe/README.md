# 井字棋

一个具有特殊规则的井字棋游戏 - 棋子会消失！

## 游戏特色

### 核心机制
- 🎮 **3x3 标准九宫格** - 经典井字棋布局
- ⚡ **棋子消失机制** - 棋盘最多同时显示 6 枚棋子
- 💫 **闪烁提示** - 即将消失的棋子会闪烁警告
- 🤖 **AI 对战** - 三种难度：简单、中等、困难

### 游戏规则
1. 率先连成 3 子获胜
2. 棋盘最多同时显示 **6 枚最新棋子**
3. **第 7 枚棋子落下时**，第 1 枚棋子消失（先进先出）
4. 即将消失的棋子会**闪烁提示**
5. 横向、纵向或对角线形成三枚连续相同棋子即获胜

## 游戏玩法

### 基本操作
- 点击空格子落子
- 你是 **X**（玩家），电脑是 **O**（AI）
- 点击"重新开始"可以开始新游戏
- 点击"难度"按钮切换 AI 难度

### 策略提示
由于棋子会消失，这个游戏比传统井字棋更有策略性：

1. **观察闪烁的棋子** - 它即将消失，不要依赖它来获胜
2. **控制节奏** - 有时候不需要急于落子
3. **多重威胁** - 尝试同时制造多个获胜威胁
4. **阻断对手** - 注意对手可能利用即将消失的棋子

### AI 难度

| 难度 | 说明 |
|------|------|
| 简单 | 随机落子，适合新手 |
| 中等 | 会进攻和防守，但有时会犯错 |
| 困难 | 策略性落子，很难击败 |
